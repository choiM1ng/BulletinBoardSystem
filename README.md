# 📋 JSP 커뮤니티 게시판  
Bootstrap과 JSP를 활용하여 구현한 간단한 커뮤니티 게시판 웹사이트입니다.  
인프런 "누구나 따라하면서 배우는 JSP 커뮤니티 게시판 만들기" 강의를 참고하여 제작했습니다.

## 📅 개발 기간  
- **2025.05.12. ~ 2025.05.18.**  


## ⚙️ 기술 스택  

- 언어: Java 17, JSP
- 프레임워크: Bootstrap
- 개발 환경: Eclipse
- 데이터베이스: MySQL
- JDK 버전: 21

## 🚀 주요 기능  

- **회원 관리:** 회원가입, 로그인, 로그아웃
- **게시글 관리:** 게시글 작성, 수정, 삭제
- **게시글 조회:** 게시글 목록 조회 및 상세 조회
